Título e descrição — "Datador de Fluxo de Entregas" — sistema de registro e controle de entrada e saída de caixas em um ambiente logístico/estoque, desenvolvido em Python.
Funcionalidades — as 4 opções do menu: registrar entrada de caixas (com horário, categoria, produto, quantidade, peso e imposto), registrar saída, visualizar relatório de entrada e relatório de saída.
Como executar — requisitos (Python 3.x instalado), comando para rodar (python nome_do_arquivo.py) e como navegar pelo menu.
Exemplo de uso — um fluxo rápido mostrando: digitar 1 → preencher os dados → digitar 3 → ver o relatório gerado.
Estrutura dos dados registrados — mostrar o que cada registro de entrada guarda (horário, categoria, produto, qtd, peso, imposto, valor do imposto) e o que o de saída guarda (horário, categoria, produto, qtd).
Tratamento de erros — mencionar que o programa valida entradas inválidas de quantidade (não-inteiro) e valor de imposto (não-numérico), além de entradas inválidas no campo sim/não.
Limitações conhecidas — dados ficam apenas em memória (sem banco de dados ou arquivo), ou seja, são perdidos ao encerrar o programa. Isso é importante documentar.
Tecnologias usadas — Python 3, sem bibliotecas externas.
